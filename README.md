# PipeAndFilter

Make sure that whichever dataset is being used, that it is placed in the Lab1/DataSets directory and named "FlightData.dat".

In order to run program, change focus to either Lab1/Sample, Lab1/Problem1, or Lab1/Problem2.

Compile code by running "javac \*.java"

Then run "java Plumber" for Lab/Sample,
         "java PlumberProblem1" for Lab/Problem1,
         "java PlumberProblem2" for Lab/Problem2.
         
Output files will be created within the directories of each problem.
